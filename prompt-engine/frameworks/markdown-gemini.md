# Framework: Markdown (Para Gemini e IAs de Texto)

## Quando Usar
Gemini 2.5 Pro, Gemini Flash, Mistral — tarefas de documentação, pesquisa, análise.

## Estrutura

```markdown
# Tarefa: {título}

## Contexto
{projeto, stack, módulo atual}

## O Que Fazer
{descrição clara da tarefa}

## Requisitos
- {requisito 1}
- {requisito N}

## Formato de Saída
{formato esperado — markdown, JSON, lista, etc.}

## Restrições
- {restrição 1}
- {restrição N}
```

## Notas
- Gemini trabalha bem com Markdown estruturado
- Bom para tarefas com contexto longo (até 1M tokens)
- Incluir exemplos de output esperado quando possível
