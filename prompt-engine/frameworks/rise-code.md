# Framework: RISE (Para IAs de Código)

## Quando Usar
IAs focadas em geração de código: Codestral, DeepSeek V3, DeepSeek R1.

## Estrutura

```
Role: {papel técnico específico}

Instructions:
{descrição da tarefa, direto ao ponto}

Steps:
1. {passo 1}
2. {passo 2}
N. {passo N}

Expected Output:
- Arquivo: {caminho}
- Formato: {linguagem/framework}
- Validações: {lista}

Constraints:
- {restrição 1}
- {restrição N}
```

## Notas
- IAs de código preferem instruções diretas e objetivas
- Menos contexto narrativo, mais especificação técnica
- Incluir exemplos de código quando possível
- Sempre especificar linguagem e framework
