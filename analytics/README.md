# Analytics

Esta pasta armazena métricas de custo, qualidade e performance.
Estrutura criada automaticamente por `/ouro:novo-projeto`.

```
analytics/
├── dashboard.json      ← Métricas agregadas
├── sessoes/            ← Log por sessão
├── fases/              ← Métricas por fase
├── prompts/            ← Histórico de prompts
│   └── historico.json
├── ias/                ← Performance por IA
└── quick/              ← Tarefas ad-hoc
```
